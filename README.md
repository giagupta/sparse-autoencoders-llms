# Archetypal SAEs for LLMs

Replicating [Fel et al. (2025) "Archetypal SAE"](https://arxiv.org/abs/2502.12892) in the language model regime (GPT-2).

Compares **Relaxed Archetypal SAE (RA-SAE)** vs **Standard TopK SAE** on dictionary stability, reconstruction quality, and monosemanticity.

See [README_comparison.md](README_comparison.md) for full details, method description, and how to run.
